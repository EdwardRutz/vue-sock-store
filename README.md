# vue-sock-store

> Demo of a basic Vue.JS app using a basic store front to sell socks.
- Creating a Vue instance and app from scratch with JavaScript, HTML, CSS



## Dependencies

- Vue.JS via link, ` https://unpkg.com/vue `
- JavaScript, HTML, CSS


## Project setup

```
npm install
npm run serve
npm run build
npm run lint
```
